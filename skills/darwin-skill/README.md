# Darwin Skill（达尔文.skill）

自主技能优化器，受 Karpathy autoresearch 启发。

## 核心功能

- **8维度评分**（满分100）：结构60分 + 效果40分
- **爬山算法**：自动迭代改进，只保留改进、自动回滚退步
- **Git版本控制**：所有改动可追溯、可回滚
- **实测验证**：spawn子agent跑测试prompt，独立评分
- **成果卡片**：3种风格可视化展示
