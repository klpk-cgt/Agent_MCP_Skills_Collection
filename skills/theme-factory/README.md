# Theme Factory

主题工厂：为任何作品（幻灯片、文档、报告、HTML页面）快速应用统一视觉风格。

## 核心功能

- 10 个预设专业主题（颜色+字体配对）
- 每个主题包含完整配色方案和字体组合
- 支持一键应用到幻灯片/文档/HTML
- 支持自定义创建新主题

## 10 大主题

1. Ocean Depths - 专业海洋风
2. Sunset Boulevard - 温暖日落色
3. Forest Canopy - 自然大地色
4. Modern Minimalist - 现代极简灰
5. Golden Hour - 浓郁秋色
6. Arctic Frost - 清爽冬日
7. Desert Rose - 柔和沙漠玫瑰
8. Tech Innovation - 科技美学
9. Botanical Garden - 清新植物园
10. Midnight Galaxy - 深邃星空
