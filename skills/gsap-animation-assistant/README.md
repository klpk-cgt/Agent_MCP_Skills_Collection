# GSAP 动画开发助手

GSAP 动画开发助手，帮助生成和审查前端动效代码。

## 核心功能

- **8个参考文档**：core / timeline / scrolltrigger / plugins / utils / react / performance / frameworks
- **精确路由表**：按用户需求自动导航到对应参考文件
- **4种框架支持**：React、Vue、Svelte、Vanilla JS
- **代码审查**：检查清理逻辑、插件注册、布局抖动、无障碍

## 触发词

`帮我做一个页面动效` `用 GSAP 实现动画` `滚动触发动画` `React/Vue/Svelte 组件加动画` `timeline 编排动画` `动画代码性能检查`
