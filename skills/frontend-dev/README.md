# Frontend Studio

全栈前端开发，结合高品质 UI 设计、电影级动画、AI 生成的媒体资源、有说服力的文案和视觉艺术。

## 五大能力

1. **Design Engineering** — 3 个可调旋钮（DESIGN_VARIANCE / MOTION_INTENSITY / VISUAL_DENSITY）
2. **Motion Engine** — 工具选择矩阵（Framer Motion / GSAP+ScrollTrigger / Lottie / Three.js / CSS）
3. **Asset Generation** — MiniMax API（图片/视频/音频/TTS）
4. **Copywriting** — 文案框架（AIDA/PAS/FAB）、标题公式、CTA 公式
5. **Visual Art** — p5.js 交互艺术 / 静态输出

## 6 阶段工作流

1. 设计架构 → 2. 动效架构 → 3. 资源生成 → 4. 文案 → 5. 构建 UI → 6. 质量门

## 设计旋钮

| 旋钮 | 默认值 | 范围 |
|------|--------|------|
| DESIGN_VARIANCE | 8 | 1=对称, 10=不对称 |
| MOTION_INTENSITY | 6 | 1=静态, 10=电影级 |
| VISUAL_DENSITY | 4 | 1=空灵, 10=密集 |

## 触发词

构建 landing pages、营销站点、产品页、仪表盘、生成媒体资源、写转化文案、生成艺术、电影级滚动动画
