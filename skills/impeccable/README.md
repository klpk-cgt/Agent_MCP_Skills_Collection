# Impeccable — 高品质前端设计工具集

创建独特、生产级的前端界面设计，避免泛 AI 审美。

## 核心能力

- **20+ 设计场景**：视觉风格调整、布局排版、动效体验、质量保障、设计系统化
- **AI Slop Test**：判断界面是否泛 AI 审美
- **DO/DON'T 速查**：字体、色彩、空间、动效、交互、响应、文案全方位指南
- **10 个设计领域深度参考**：排版、色彩、空间、动效、交互、响应式、UX文案、启发式评分、用户画像、认知负载

## 触发词

`界面设计` `前端设计` `响应式` `动效` `配色` `设计评审` `排版` `设计系统` `视觉冲击` `降噪` `打磨` `炫酷效果` `无障碍` `用户体验` 等

## 核心原则

- 禁止 AI 审美（Inter 字体、纯黑 #000、AI 紫蓝色）
- 每次设计都不同、意想不到的创意选择
- 选择大胆的美学方向（极简/混沌极大化/复古未来/有机自然/野兽派等）
