# Grill Me

深度追问式方案审查——对用户的计划或设计进行逐层拆解，逐个解决决策树中的每个分支，直到达成共识。

## 核心功能

- 列出顶层决策分支（3-6项）
- 从最基础的开始逐层追问
- 每个问题先给出推荐答案再提问
- 逐个解决决策树中的每个分支
- 可用 Read/Grep 探索代码库回答技术问题

## 触发词

`grill me` `challenge my thinking` `stress-test`

## 与 grill-with-docs 的区别

- **grill-me**：纯对话式审查，不读/更新文档，用于概念性计划
- **grill-with-docs**：锚定项目 CONTEXT.md/ADRs 审查，用于已有领域模型的项目
